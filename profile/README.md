## Overview

<a href="#">CoinLedger</a> automates cryptocurrency tax calculation and portfolio tracking for investors who trade across multiple exchanges and wallets. The application imports complete transaction histories, categorizes each transaction type correctly, applies the appropriate tax treatment, and generates the reports that accountants and tax filings require.

The <a href="#">transaction import engine</a> connects to hundreds of exchanges including Coinbase, Binance, Kraken, Gemini, and others through read-only API access. Historical transaction data imports completely without manual entry. <a href="#">CSV upload support</a> handles exchanges not covered by direct API integration, accepting transaction exports from virtually any platform. Wallet address tracking imports on-chain transaction data directly from blockchain explorers for activity that never touched an exchange.

<a href="#">Automatic transaction categorization</a> identifies trades, transfers, income events, and other transaction types without manual review. Trades calculate cost basis and realized gains or losses. Transfers between your own wallets pass through without triggering taxable events. <a href="#">Income recognition</a> records staking rewards, mining proceeds, and airdrop receipts at fair market value on the date received.

The <a href="#">portfolio dashboard</a> displays current holdings, total value, unrealized gains and losses, and performance over time. See how your portfolio has evolved since initial investment with interactive charts. <a href="#">Gain and loss summary</a> breaks down realized results by asset, time period, and transaction type. This reporting provides both the tax information needed for filing and the investment performance data useful for portfolio management decisions.

## Key Features

- <a href="#">Exchange API integration</a> imports complete transaction history from hundreds of major crypto platforms
- <a href="#">Wallet address tracking</a> pulls on-chain data directly from blockchain explorers for all activity
- <a href="#">Automatic categorization</a> identifies trades transfers income events and non-taxable movements correctly
- <a href="#">DeFi protocol support</a> classifies yield farming liquidity pools lending and decentralized exchange swaps
- <a href="#">NFT gain and loss tracking</a> calculates capital results and royalty income from token trading activity
- <a href="#">Multiple cost basis methods</a> apply FIFO LIFO HIFO with jurisdiction-aware default selection
- <a href="#">Portfolio performance dashboard</a> shows current holdings value unrealized gains and historical charts
- <a href="#">Tax loss harvesting analysis</a> identifies offsetting loss opportunities before the tax year closes
- <a href="#">Jurisdiction-specific reports</a> generate filing-ready output for US UK Canada Australia and more
